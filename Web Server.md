
A web server handles HTTP and TCP connections, manages the resources served by the web server, and provides administrative features to configure, control, and enhance the web server.

# Web Resources

- **Static resources**: directly accessed and do not require additional processing.
- **Dynamic resources**: Application server runs the processing steps (search, transaction, ...)

# HTTP Connections Handling

- **Single-Process**: One request at a time
- **Multi-Process**: Multiple Processes -> Multiple requests in parallel
- **Multiplexing**: One process with multiple threads
