
**Apache** is a web server.

# FEATURES

- **Modular design**: compromised by several core modules
- **Dynamic Shared Object (DSO)**: Load/Unload DSO modules.
- **Multi-Processing Module (MPM)**: Configure the working mode of responding to multiple requests

**code modules** must be statically compiled during Apache installation, such as:

- **mod_so**: is used to process configuration files
- **mod_core**: is used to dynamically load other modules
